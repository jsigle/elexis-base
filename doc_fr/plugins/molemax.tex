MoleMax avait été developpé principalement pour des dermtologues mais peut aussi être d'intérêt pour un médecin de premier recours. Il s'agit d'un plugin pour la documentation de l'évolution des altérations de la peau.

 On peut intégrer les images digitales et les comparer de façon systématique. Une déscription détaillée et instructions se trouve sur le site \href{http://www.rgw.ch/elexis/dox/molemax.pdf}{ici (PDF)}.
MoleMax utilise Elexis 1.1.1 et peut être acheté (y inclus les mises à jour pendant une année) pour CHF 460.- .  Les abonnés d'un des notres contrats de maintenance annuaires peuvent l'avoir gratuitement. Pour des commandes et plus d' informations : admin (at) elexis.ch.
