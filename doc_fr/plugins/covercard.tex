\section{Covercard{\textregistered}}
\index{CovercardTM}
Ce plugin rend service  pour l'utilisation des cartes des patients fournies par les assurances compatibles au système Covercard{\textregistered}. Il suffit de passer la carte à travers le lecteur réspectivement d'introduire le numéro de la Covercard pour pouvoir introduire les données de l'assuré de façon automatisée. Vous pouvez trouver des informations supplémentaires sous   \href{http://www.elexis.ch/files/covercard.pdf}{Instructions pour la Covercard (PDF)}.
Ce plugin Elexis-Covercard nécessite Elexis > version 1.1.1 et un  ASAS-Client avec abonnement Covercard.
Das Covercard-Plugin kostet CHF 94.-.

Ce plugin est gratuit pour tous ceux qui étaient déjà avant le 12.8.2007 sur la mailing-liste des utilisateurs de Elexis. 
Pour une commande veuillez vous adresser à  admin (at) elexis.ch.
