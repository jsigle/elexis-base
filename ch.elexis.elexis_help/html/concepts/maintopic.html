Programmphilosophie

Leichtigkeit. Wo andere Programme Sie dazu zwingen, ihre Bedienung zu erlernen, ist Elexis von vornherein so ausgelegt, dass es ihrem gewohnten Arbeitsfluss entgegenkommt, so dass Sie ohne besondere Schulung sofort mit der Arbeit anfangen können - das Erlernen einiger weniger Grundkonzepte genügt

Flexibilität. Nicht Sie müssen sich an den Computer anpassen, sondern der Computer soll sich gefälligst an Sie anpassen. Diese Philosophie ist ein ganz zentrales Element bei der Entwicklung von Praxisdesktop. Nicht nur das Aussehen der Benutzeroberläche, sondern auch die Abläufe "unter der Haube" sind in weitestem Rahmen individuell einstellbar. Und last but not least können Sie Elexis auf verschiedensten Betriebssystemen einsetzen. Egal, ob Sie auf Macintosh, Windows oder Linux schwören, Elexis läuft auf Ihrem System.

Zuverlässigkeit. Elexis baut auf bewährten und robusten Komponenten auf. Eclipse ist ein von IBM und anderen grossen Firmen entwickeltes und auch in eigenen Produkten eingesetztes Framework.
