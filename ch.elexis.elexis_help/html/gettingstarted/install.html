Installation

Installation unter Microsoft Windows 2000 oder neuer

  • Führen Sie das Installationsprogramm "Elexis.exe" aus und folgen Sie den Anweisungen am Bildschirm. Danach finden Sie auf dem Desktop ein neues Icon "Elexis", mit dem Sie das Program starten können

Installation unter Linux oder Macintosh

  • Entpacken Sie die Datei Elexis.zip resp. Elexis.tar.bz in Ihr Programmverzeichnis.
  • Im ordner "Elexis" finden Sie eine Datei "Elexis.jar". Diese Datei kann mit der Java virtual machine gestartet werden und lädt das Hauptprogramm nach
