Was ist mit "modular" gemeint?

Die Ansprüche sind verschieden. Der Eine braucht Anbindung externer Geräte, eine Agenda, psychologische Tests, eine Ablage für Bilder, der andere braucht das nicht. Elexis bietet alle wesentlichen Grundfunktionen vom Staert weg. Das Programm ist aber so konzipiert, dass zusätzlicher Funktionen sehr einfach als Zusatzmodule entwickelt werden können, welche nahtlos in die bestehende Installation integriert werden können.

Die Module sowohl des Grundsystems als auch allfällige Zuatzmodule können ausserdem unabhängig voneinander weiterentwickelt werden. Die halbautomatische Update-Funktion von Elexis sorgt dafür, dass jedes Mdul bei Bedarf auf den neuesten Stand gebracht wird, ohne dass jemals eine Neuinstallation von Elexis erforderlich ist.
