Systemvoraussetzungen

Allgemeine Voraussetzungen

Elexis kann auf den meisten aktuellen Computern mit dem Betriebssystem Windows 2000 oder neuer, Apple Macintosh OS X oder neuer, Linux ab Kernel 2.6 und mit KDE oder Gnome-Oberfläche und anderen Systemen installiert werden. Achten Sie darauf, die zu Ihrem System passende Version zu verwenden.

Weitere Hardware-Voraussetzungen

Zentraler Server bei Server/Client - Versionen bzw. Arbeitsplatz bei Einzelplatzversionen

  • Mindestens 512MB RAM, besser 768 MB
  • Prozessor mit mindestens 1.5 GHz Taktfrequenz
  • Festplatte mind mindestens 1GB freiem Platz

Client-Computer bei Server/Client - Versionen

  • Mindestens 256 MB RAM, besser 512 MB
  • Prozessor mit mindestens 1GHz Taktfrequenz
  • Festplatte mit mindestens 100 MB freiem Platz

Software-Voraussetzungen

  • Ein aktuelles Betriebssystem, s. unter Allgemeine Vorausetzungen
  • Für den Server bzw. Einzelplatz: Ein SQL- und JDBC-fähiger Datenbankserver. Beispielsweise Microsoft SQL-Server, PostgreSQL, MySQL, IBM Cloudscape, HSQLDB, Ingres, Interbase, Firebird, Adabas und andere. Wir bevorzugen HSQLDB für die Einzelplatzversion und PostgreSQL für die Client/Server-Version, aber das ist keine Bedingung.
